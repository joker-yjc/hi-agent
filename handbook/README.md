# 前端开发者的 AI 开发手册

> 一份"前端视角"的 LLM/Agent 开发指南——既有验证过的可靠知识点，也有如何在 TypeScript 技术栈里落地的代码思路。
> 基于 [Vercel AI SDK v6](https://sdk.vercel.ai) + [Next.js](https://nextjs.org) 生态。

---

## 这是什么

**不是一份纯教材**（概念→原理→练习），**也不是一份 API 文档**（照着抄就行）。

这是我在学习 AI 开发过程中踩坑之后整理的实战笔记。每章解决两个问题：
1. **这个概念到底是什么**（省去你从搜索引擎结果中鉴别真伪的成本）
2. **怎么在你的技术栈里落地**（前端视角的代码骨架和选型建议）

## 适用人群

- 前端开发者想学 AI 应用开发，但不知道从哪入手
- 网上资料太多太杂，分不清哪些可信、哪些过时
- 想用 TypeScript/JavaScript 技术栈，不想从头学 Python

## 每章格式

| 板块 | 解决什么问题 |
|------|------------|
| **概念速览** | 这是什么东西？核心要点一句话 + 与容易混淆概念的区别 |
| **技术选型** | 前端技术栈下应该用什么库/工具？优缺点是什么？ |
| **代码骨架** | 最重要 10-20 行代码，可以直接抄的骨架 |
| **实战建议** | 对应学习计划的 Day 任务，实现思路和关键决策点 |
| **踩坑记录** | 我亲自踩过的坑 + 怎么绕过去 |
| **延伸阅读** | 只列验证过的可信资料（官方文档/源码/博客） |

## 配套的代码项目

本手册的知识点均有对应可运行代码，位于同仓库的各 `stage-*/` 目录：

| 阶段 | 项目 | 内容 |
|------|------|------|
| Stage 2 | `stage-2-api/` | 5 个脚本：generateText、streamText、多轮对话、结构化输出、Ollama |
| Stage 3 | `stage-3-chat-app/` | Next.js Chat 应用：useChat、Markdown 渲染、多模型、对话历史 |
| Stage 4 | `stage-4-rag/` | 6 个脚本：Embedding、分块、索引、搜索、完整 RAG 管道 |
| Stage 4 App | `stage-4-rag-app/` | Next.js RAG Web：文档上传、向量存储、检索问答 |
| Stage 5 | `stage-5-agent/` | 7 个 Agent 脚本 + 工具模块：Tool Calling、三种记忆模式 |
| 共享库 | `packages/shared-utils/` | 模型 Provider 统一抽象：qwen/deepseek/openai/ollama |
